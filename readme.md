# Microsoft Agentic AI Architect Program вЂ“ June 2026

## Overview

This repository contains training materials, architecture diagrams, hands-on labs, exercises, reference guides, and supporting resources for the **Microsoft Agentic AI Architect Program**.

The program is designed for AI Architects, Solution Architects, Business Architects, Enterprise Architects, Technical Leads, Consultants, and Business Transformation teams responsible for designing and governing enterprise AI solutions using Microsoft technologies.

Participants will learn how to design, govern, orchestrate, secure, and operationalize Agentic AI solutions using:

* Microsoft 365 Copilot
* Copilot Studio
* Azure AI Foundry
* Microsoft Fabric
* Dynamics 365
* Power Platform
* Azure AI Services

---

# Learning Outcomes

By the end of this program, participants will be able to:

* Perform AI Business Architecture assessments
* Evaluate enterprise AI opportunities
* Design multi-agent architectures
* Build Copilot Studio solutions
* Design grounding and retrieval strategies
* Implement AI governance and security controls
* Develop ROI and TCO models
* Design orchestration across Microsoft platforms
* Implement monitoring and telemetry strategies
* Apply Responsible AI principles
